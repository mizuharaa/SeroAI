## ✨ 主な機能

### 🎯 **5軸検出システム**
- **モーション/時間的安定性** (50%の重み): フレーム間の不整合、オプティカルフロー異常、時間的アーティファクトを検出
- **生物学的/物理的現実性** (20%の重み): 顔のランドマーク、まばたきパターン、解剖学的整合性、体の動きを分析
- **シーンと照明の論理** (15%の重み): オブジェクトの持続性、物理的整合性、照明の一貫性、ショット境界を検証
- **テクスチャと周波数アーティファクト** (10%の重み): GAN指紋、スペクトルパターン、圧縮アーティファクト、テクスチャの不整合を識別
- **透かしと出所** (5-50%の重み): 検証されたAIモデルの透かしの視覚的ロゴマッチング (Sora, Gemini, Pika, Luma, Runway, HeyGen, D-ID)

### 🔍 **高度な検出機能**
- **視覚的ロゴマッチング**: 検証された透かし検出のためのテンプレートマッチング、ORB特徴マッチング、ヒストグラム比較、SSIM
- **統合推論**: 複数の弱い信号をインテリジェントに組み合わせて、誤検出を減らし、信頼度を向上
- **意味論的不可能性検出**: 論理的に不可能なシナリオにフラグを立てる (例: 新しい映像での故人セレブ)
- **動的重み調整**: 検証されたAIロゴが検出されると、透かし主導の重み(50%)に自動切り替え
- **品質ゲート**: 誤検出を防ぐために低品質メディアを事前フィルタリング

### 🎨 **モダンなWebインターフェース**

## 🚀 クイックスタート

### 前提条件

- **Python 3.9+** (3.10+推奨)
- **Node.js 18+** および npm
- **FFmpeg** (ビデオ処理用)
- **Tesseract OCR** (オプション、テキストベースの透かし検出用)

### インストール

```bash
# 1. リポジトリをクローン
git clone https://github.com/<your-org-or-user>/SeroAI.git
cd SeroAI

# 2. 仮想環境を作成してアクティブ化
python -m venv .venv
# Windows (PowerShell):
.\.venv\Scripts\Activate.ps1
# macOS/Linux:
source .venv/bin/activate

# 3. Python依存関係をインストール
python -m pip install --upgrade pip
python -m pip install -r requirements.txt

# 4. フロントエンド依存関係をインストール
cd webui
npm ci
npm run build
cd ..

# 5. サーバーを起動
python app.py
```

サーバーは `http://localhost:5000` で起動します。

### システム依存関係

**Windows (PowerShell)**:
```powershell
winget install ffmpeg
winget install tesseract  # オプション
```

**macOS**:
```bash
brew install ffmpeg
brew install tesseract  # オプション
```

**Linux (Ubuntu/Debian)**:
```bash
sudo apt-get update
sudo apt-get install ffmpeg tesseract-ocr  # オプション
```
