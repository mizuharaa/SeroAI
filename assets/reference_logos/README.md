# Reference Logo Assets

This directory should contain transparent PNG/SVG logos for AI model providers.

## Required Logos

Place the following logo files here:

- `sora_logo.png` - OpenAI Sora logo (knot emblem + wordmark)
- `gemini_logo.png` - Google Gemini logo (blue gradient wordmark with star/spark icon)
- `pika_logo.png` - Pika Labs logo
- `luma_logo.png` - Luma AI logo (blue/purple "L" shard logo)
- `runway_logo.png` - Runway logo
- `heygen_logo.png` - HeyGen logo
- `did_logo.png` - D-ID logo

## Logo Requirements

- **Format**: PNG with transparent background (or SVG)
- **Quality**: High resolution, clear brand shapes
- **Official**: Use official, stable logo assets from each provider
- **Transparency**: Alpha channel for proper matching

## Note

If logos are not found, the system will fall back to text-based detection.
For production use, ensure all reference logos are present.
